# Project Execution

- This project was made using versi√≥n 17 of nodeJs, so it may not work properly if a lower version is used. To ensure a proper execution, please use nodeJs 17 or higher version.

### Steps
- Clone repository into your local
```js
git clone https://github.com/JaviMunita/pokedex.git
```

- Making sure you are in the pokedex folder, install dependencies
```js
npm install
```

- Run project to see it in the localhost:5173 port
```js
npm run dev
```
